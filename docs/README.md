# RogueZ 文档（`docs/`）

本目录存放与游戏内容、术语相关的可读文档。

| 文档 | 说明 |
|------|------|
| [GLOSSARY.md](./GLOSSARY.md) | 名词中英文对照（与 `i18n.js` 同步） |
| [ECONOMY.md](./ECONOMY.md) | 终局分数、击杀分、金币转化率与局外经济备忘 |

## 维护约定

在合并或发布前，若改动涉及以下内容，请**同步更新 `docs/` 中相应文件**（至少检查 [GLOSSARY.md](./GLOSSARY.md) 是否需改）：

- 新增或修改 **UI 文案**、**多语言键**（[`i18n.js`](../i18n.js)）
- 新增或重命名 **游戏实体**（武器、敌人、宠物、拾取物类型等，见 [`game.js`](../game.js)）
- 新增 **玩家可见的系统/操作说明**（设置、操作方式、图鉴等）
- 修改 **结算分、金币换算、商店/转盘价格** 时，更新 [ECONOMY.md](./ECONOMY.md)

详细规则与表格结构见 [GLOSSARY.md](./GLOSSARY.md) 文首「文档用途与维护」一节。
